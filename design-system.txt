TYPOGRAPHY
----------
PRIMARY FONT (Headings):
• Font Family: 'Poppins', sans-serif
• Usage: All headings (h1-h6), brand text, important CTAs
• Weights Available: 400 (Regular), 600 (SemiBold), 800 (ExtraBold)

BRAND FONT (Logo/Brand):
• Font Family: 'Poppins', sans-serif
• Font Weight: 800 (ExtraBold)
• Usage: Logo text "SapSap", brand signatures
• Class: .brand-font

BODY FONT:
• Font Family: 'Inter', sans-serif
• Usage: Body text, descriptions, form inputs
• Weights: 400 (Regular), 500 (Medium), 600 (SemiBold)

FONT SCALE & HIERARCHY:
• Hero Title: text-5xl md:text-7xl (80px-112px)
• Section Headers: text-4xl md:text-5xl (56px-72px)
• Subsection Headers: text-2xl md:text-3xl (24px-32px)
• Body Large: text-xl md:text-2xl (20px-24px)
• Body Regular: text-base (16px)
• Small Text: text-sm (14px)

GLOW EFFECTS
-----------
• Red Glow: 0 0 20px hsla(357, 89%, 48%, 0.5)
• Cyan Glow: 0 0 20px hsla(171, 100%, 38%, 0.5)
• Classes: .hover-glow-red, .hover-glow-cyan

ANIMATION GUIDELINES
-------------------
• Transition Duration: 300ms (standard)
• Hover Effects: Scale 1.05 for logos, glow for buttons
• Story Carousel: Linear infinite animation, 10s-30s duration
• Responsive Animation: Reduced motion support included

BREAKPOINTS
-----------
• Mobile: < 768px
• Tablet: 768px - 1024px
• Desktop: > 1024px
• Large Desktop: > 1440px

ACCESSIBILITY
------------
• Color Contrast: All text meets WCAG AA standards
• Focus States: Visible focus rings on interactive elements
• Reduced Motion: Respects prefers-reduced-motion
• Alt Text: All images include descriptive alt attributes

CONTENT GUIDELINES
-----------------
HERO SECTION:
• Main headline should be inspiring and action-oriented
• Subtitle explains value proposition clearly
• CTA should create urgency ("Reserva tu lugar en el prelanzamiento")

STORY COVERS:
• Descriptions should be 2 lines maximum on hover
• Focus on experience and emotion rather than technical details
• Use active voice and engaging language

MULTI-LANGUAGE SUPPORT
---------------------
• Supported Languages: Spanish (default), English, French, Portuguese
• Auto-detection based on IP geolocation
• Manual language switcher in top-right corner
• Flag icons for visual language identification

VIDEO ASSETS
-----------
• Hero Video: 
  - Desktop: 1.9MB optimized
  - Mobile: 1.3MB optimized
  - Poster: 66KB for lazy loading
  - Format: MP4, H.264 codec

• Social Media Videos:
  - Format: 1080x1920 (vertical)
  - Duration: 5 seconds
  - Size: < 50KB for optimal loading
  - Include brand logo and hook message

BRAND VOICE & MESSAGING
----------------------
• Tone: Inspiring, global, accessible, modern
• Key Messages:
  - "Transform experiences into stories"
  - "Share your journey with the world"
  - "Your first viral story is waiting"
• Avoid: Technical jargon, complex language, lengthy explanations

DESIGN PRINCIPLES
----------------
1. MINIMALISM: Clean, focused design with plenty of whitespace
2. CONTRAST: Strong color contrast for readability and impact
3. HIERARCHY: Clear visual hierarchy through typography and spacing
4. RESPONSIVENESS: Mobile-first approach with scalable design
5. PERFORMANCE: Optimized assets and lazy loading for fast experiences
6. ACCESSIBILITY: Inclusive design for all users
